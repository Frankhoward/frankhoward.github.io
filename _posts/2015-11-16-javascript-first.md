# 写在前面 #
捯饬了一周，终于磕磕碰碰、踉踉跄跄地，问世了。来得不算早也不算晚，总算是有了一个私人博客，不拘泥于内容，在这里，可以写自己想记录的东西。

很多东西都是摸着石头过河，比葫芦画瓢搞出来的，可是不管是黑猫警长还是蓝猫淘气三千问，能抓老鼠就是好猫。今后还要慢慢摸索，把自己想表现的东西一一做出来。
